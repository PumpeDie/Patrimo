# Patrimo

Patrimo is an open-source wealth management application designed to simplify the process of tracking and optimizing your financial assets.

## Features

- **Comprehensive Asset Tracking:** Monitor various asset classes, including investments, real estate, and savings.
- **Performance Analysis:** Gain insights into the growth and performance of your portfolio over time.
- **User Authentication:** Secure login and registration system to protect your financial data.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PumpeDie/patrimo.git
   ```
